vcard2FritzXML Get your FRITZ!Box synchronised.

vcard2fritzXML is a tool written in Java to convert a vcard file to a FRITZ!Box address book import file. It's under the GPLv2 license.
FRITZ!Box is a popular DSL router in Germany by AVM.
 
More informations is on https://github.com/berkholz/vcard2fritzXML.

PaMooBa Parallize your Moodle Backups.

PaMooBa stands for parallized Moodle backup. First I developed it as a proof of concept tool while working at the HTW for speeding up the backup of large Moodle installations. After leaving the HTW I wanted to share the tool as open source, so I completly redeveloped the tool with a different configuration format and database support for mysql and postgresql.
 
For more informations see https://github.com/berkholz/pamooba.
